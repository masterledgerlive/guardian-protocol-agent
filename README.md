# ⬡ Guardian Protocol Agent

AI-powered wallet agent built on Base Sepolia testnet.
Powered by Coinbase AgentKit and Claude AI.

## What This Agent Does

1. Creates a Guardian wallet on Base Sepolia
2. Checks balance and requests testnet funds
3. Sends a real test transaction with on-chain proof

## Stack

- Runtime: Node.js
- AI Brain: Claude (Anthropic)
- Wallet: Coinbase AgentKit
- Chain: Base Sepolia testnet
- Memory: wallet_data.json

## Setup

1. Clone this repository
2. Run npm install
3. Create .env file with your keys:

CDP_API_KEY_NAME=your_key_name
CDP_API_KEY_PRIVATE_KEY=your_private_key
NETWORK_ID=base-sepolia
ANTHROPIC_API_KEY=your_anthropic_key

4. Run node agent.js

## Keys Required

- Coinbase CDP API keys from portal.cdp.coinbase.com
- Anthropic API key from console.anthropic.com
